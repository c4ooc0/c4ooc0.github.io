第0期 DVWA安装部署（Windows）

发表于 2022-11-10 本文字数： 阅读时长 ≈ NaN:aN

Damn Vulnerable Web Application（DVWA）是一个非常脆弱的PHP/MySQL Web应用程序。它的主要目标是帮助安全专业人员在合法环境中测试他们的技能和工具，帮助web开发人员更好地理解保护web应用程序的过程，并帮助学生和教师在受控教室环境中学习web应用程序安全。 看DVWA首页似乎推荐部署在虚拟机里，有机会改一下

第一步 下载phpstudy并安装

https://www.xp.cn/download.html 解压后进行安装，一般装在D盘问题不大。 安装完成后，浏览器访问127.0.0.1，看到如下界面即安装成功。

第二步 下载DVWA源码

https://github.com/digininja/DVWA 或者使用命令： git clone https://github.com/digininja/DVWA.git

打开phpstudy网站的根目录，在phpstudy的安装目录下，路径一般是...\phpstudy_pro\WWW。 将解压后的DVWA文件夹复制到该目录。方便期间，DVWA-master文件可以改名为DVWA，以下以改名为前提进行演示。

DVWA->config->config.inc.php.dist文件，修改后缀名为ini.php。

打开该文件，修改数据库密码。 首先查看phpstudy中的数据库密码，鼠标移到密码处就显示了。

将db_user改为root，db_password根据phpstudy显示的改，我这里是123456。 )

第三步 创建数据库

浏览器访问127.0.0.1/DVWA/setup.php，找到最后，创建数据库。(这里如果失败，一般就是数据库名或者密码不对，看一下phpstudy的数据库设置就知道了)

访问http://127.0.0.1/DVWA/index.php 进行登录，用户名/密码：admin/password。登陆成功后，DVWA就可以正常使用了。
